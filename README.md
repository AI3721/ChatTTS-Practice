# MyChat
> [!TIP]
> 新手小白入门——基于ChatTTS的FastAPI+Gradio入门练习项目
## 项目运行
### 1. 启动FastAPI
```bash
cd fastapi
uvicorn main:app --port 7788
```
### 2. 启动Gradio
```bash
cd gradio
python demo.py
```
